💻4-1 C언어 핵심
★★★

포인터 핵심

int a = 10; int *p = &a; // p에 a의 주소 저장 *p = 20; // 역참조: a의 값을 20으로 변경 p + 1 // 주소 + sizeof(int) = +4바이트 // 배열과 포인터 int arr[5]; arr == &arr[0] // 배열명 = 첫 원소의 주소 arr[i] == *(arr+i) // 동일!
⚠ C언어 함정
  • 전위++ vs 후위++: a = b++는 대입 후 증가 / a = ++b는 증가 후 대입
  • sizeof("abc") = 4 (널문자 포함) / strlen("abc") = 3
  • 문자열 비교: str1 == str2는 주소 비교! → strcmp() 사용!
  • 정수 나눗셈: 5/2 = 2 (소수 버림)

자주 나오는 코드 추적

// Q: 출력 결과는? int a = 5; int b = a++; // b=5, a=6 (후위) printf("%d %d", a, b); // 6 5 // Q: 출력 결과는? int arr[] = {10,20,30,40,50}; printf("%d", *(arr + 3)); // 40 // Q: sizeof 결과는? sizeof("Hello") // 6 (H,e,l,l,o,\0) strlen("Hello") // 5 (\0 미포함)

📝 McCabe 순환복잡도 계산 공식
  • 공식 1: V(G) = 분기문(조건문) 수 + 1
  • 공식 2: V(G) = E − N + 2P
  • (E=간선수, N=노드수, P=연결컴포넌트수)
  • if 3개 + while 2개 → V(G) = 5+1 = 6
  • 복잡도 10 이하 권장
